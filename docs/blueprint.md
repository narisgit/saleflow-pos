# **App Name**: SaleFlow

## Core Features:

- Product Management: Ability to add, view, edit, and delete product details (name, price, stock).
- Point of Sale Interface: An intuitive user interface for browsing products, adding them to a transaction cart, and adjusting quantities.
- Cart & Total Calculation: Dynamically display selected items, calculate subtotal, and show the final transaction total.
- Transaction Finalization: Functionality to mark an order as complete and clear the current transaction cart, storing the order.
- Order History View: A feature to view a list of past completed transactions with basic details such as date, total, and items.
- AI-Powered Product Description Tool: A tool to generate engaging and concise product descriptions based on product name and key attributes for new inventory entries.
- Barcode Scanning: Scan product barcodes using the device camera to quickly add items to the transaction cart.
- Barcode Generation: Generate unique barcodes for new or existing products that lack them, for easier scanning and inventory management.

## Style Guidelines:

- Primary Color: A clean, professional blue (#337ACC) to convey trust and efficiency.
- Background Color: A soft, desaturated light blue (#ECF2F8) to maintain clarity and provide a calm canvas.
- Accent Color: A vibrant cyan-green (#2ED4D4) for call-to-action buttons and key interactive elements, providing a modern contrast.
- Headline and body font: 'Inter', a neutral and highly readable sans-serif, ensuring clarity for product names, prices, and numerical data.
- Utilize modern, clean, and functional line icons for navigation, product categories, and transaction actions to enhance usability.
- Employ a modular grid-based layout for product displays, with clearly defined sections for the sales cart and transaction summary to ensure easy information processing.
- Incorporate subtle and quick animations for feedback on user interactions, such as adding items to the cart or finalizing a transaction, enhancing responsiveness.
