
# SaleFlow POS - ระบบจัดการจุดขายสำหรับผู้เริ่มต้น (Learning Mode)

โปรเจกต์นี้ได้รับการตั้งค่าให้สามารถ **Deploy ได้ฟรี (แผน Spark)** เพื่อการเรียนรู้ โดยไม่ต้องผูกบัตรเครดิต

## 🚀 ทางเลือกในการนำไปใช้งาน (Deployment)

เนื่องจากตอนนี้โค้ดของคุณอยู่บน **GitHub** แล้ว คุณมี 2 ทางเลือก:

### ทางเลือกที่ 1: แบบฟรี 100% (Manual Deploy)
วิธีนี้ไม่ต้องผูกบัตรเครดิต แต่ต้องทำผ่านเครื่องคอมพิวเตอร์ของคุณ:
1. **Download Code**: ดาวน์โหลด ZIP ไฟล์จาก GitHub ลงเครื่องคอมพิวเตอร์
2. **ติดตั้งโปรแกรม**: เปิด Terminal ในโฟลเดอร์โปรเจกต์แล้วรัน:
   ```bash
   npm install
   ```
3. **Build โค้ด**:
   รันคำสั่ง:
   ```bash
   npm run build
   ```
   *ระบบจะสร้างโฟลเดอร์ชื่อ `out` ขึ้นมา*
4. **Deploy**:
   รันคำสั่ง:
   ```bash
   firebase deploy --only hosting
   ```
   *ลิงก์ที่ได้จะลงท้ายด้วย `.web.app` **สามารถนำลิงก์นี้ไปเปิดในมือถือเพื่อใช้งานจริงได้ทันที***

### ทางเลือกที่ 2: แบบอัตโนมัติ (App Hosting - ต้องใช้แผน Blaze)
วิธีนี้สะดวกที่สุด แต่ต้องผูกบัตรเครดิตใน Firebase Console:
1. ไปที่ **Firebase Console** -> **Build** -> **App Hosting**
2. เชื่อมต่อกับ Repository ใน GitHub ของคุณ
3. ระบบจะ Deploy ให้อัตโนมัติทุกครั้งที่มีการแก้โค้ด

## 📊 การจัดการข้อมูล
- ดูข้อมูลดิบ (สินค้า/ยอดขาย/พนักงาน) ได้ที่: [Firebase Console](https://console.firebase.google.com/) -> Build -> Firestore Database

## ⚠️ ข้อจำกัดของโหมดฟรี (Static)
- **AI (Genkit) จะใช้งานไม่ได้**: เนื่องจากระบบ AI ต้องการเซิร์ฟเวอร์ประมวลผล (Cloud Run)
- **ความเร็ว**: การโหลดหน้าเว็บอาจช้ากว่าแบบ App Hosting เล็กน้อย
- **ฐานข้อมูล (Firestore/Auth)**: ใช้งานได้ตามปกติ 100% (บันทึกข้อมูลได้จริง)
