# SaleFlow POS - ระบบจัดการจุดขายร้านอาหารสัตว์

ยินดีด้วย! แอปพลิเคชัน SaleFlow POS ของคุณพร้อมใช้งานแล้ว นี่คือคำแนะนำสำหรับการนำไปใช้งานจริงและการจัดการข้อมูล

## 📊 การจัดการข้อมูล (Data Management)
ข้อมูลทั้งหมดของระบบถูกจัดเก็บไว้ใน **Firebase Cloud Firestore** อย่างปลอดภัย
- **วิธีดูข้อมูล**: เข้าไปที่ [Firebase Console](https://console.firebase.google.com/)
- **เมนู**: เลือกโปรเจกต์ของคุณ -> **Firestore Database**
- **สิ่งที่คุณทำได้**: ตรวจสอบรายการขายทั้งหมด, แก้ไขสต็อกสินค้าโดยตรง, หรือลบข้อมูลพนักงานในกรณีฉุกเฉิน

## 🚀 การนำไปใช้งานจริง (Deployment)
เพื่อให้พนักงานเข้าใช้งานได้จากมือถือหรือคอมพิวเตอร์ส่วนตัว คุณต้องทำการ Deploy แอปพลิเคชัน:
1. **Firebase App Hosting**: ระบบนี้ถูกตั้งค่าให้รองรับ Next.js 15 อย่างสมบูรณ์ผ่านไฟล์ `apphosting.yaml`
2. **ขั้นตอน**:
   - เชื่อมต่อ Repository ของคุณกับ Firebase App Hosting ในหน้าคอนโซล
   - ระบบจะทำการ Build และสร้าง URL ถาวร (เช่น `https://saleflow-pos.web.app`) ให้คุณนำไปให้พนักงานใช้งาน

## 📱 การใช้งานบนมือถือ
เมื่อ Deploy สำเร็จแล้ว ให้พนักงานเปิด URL ผ่านเบราว์เซอร์บนมือถือ (Safari/Chrome) และแนะนำให้กด **"เพิ่มลงในหน้าจอโฮม" (Add to Home Screen)** เพื่อให้ใช้งานได้สะดวกเหมือนแอปพลิเคชันทั่วไป

## 🔐 ความปลอดภัย
- อย่าลืมตั้งค่า **Firebase Authentication** ในหน้าคอนโซลให้เปิดใช้งานวิธี Login แบบ Email/Password
- ตรวจสอบสิทธิ์พนักงานในเมนู "จัดการพนักงาน" ภายในแอปเป็นประจำ
